# Two Leaf

A tea-shop web app build with eleventy and nunjucks.

### Requirements

- **Node.js**: >= 18.0.0 (gebaut mit 24.11.1)
- **NPM**: >= 9.0.0 (gebaut mit 11.6.2)
- **Git**: <https://git-scm.com>

## Installation

1. clone repository
   git clone <repository-url>

2. navigate to project folder
   cd two-leaf

3. install dependencies
   npm install

### NPM Scripts

- 'npm start' -> start eleventy dev server at http://localhost:8080/

### Technologies

- [Eleventy (11ty)](https://www.11ty.dev/) - Static Site Generator
- [Nunjucks](https://mozilla.github.io/nunjucks/) - Templating
- [SCSS](https://sass-lang.com/) - Styling
- [Swiper](https://swiperjs.com/) - Slider Library
